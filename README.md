## Yêu cầu hệ thống
- Đã cài đặt Node.js
- Đã cài đặt MongoDB (nếu dùng local)

## Hướng dẫn cài đặt và chạy dự án

1. Clone mã nguồn hoặc giải nén file.
2. Mở terminal tại thư mục dự án và chạy lệnh cài đặt thư viện:
   npm install

3. Cấu hình biến môi trường:
   - Copy file `.env.example` và đổi tên thành `.env`
   - Điền các thông tin cần thiết vào file `.env` (Đặc biệt là MONGODB_URL).

4. Chạy dự án:
   npm start
